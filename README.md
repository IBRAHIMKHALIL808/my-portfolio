# 🚀 Personal Portfolio Website

A colorful and responsive personal portfolio website built as part of the **Internet Programming** course project at **Altınbaş University, Software Engineering Department**.

---

## 📌 Project Overview

This project is a single-page portfolio website designed to showcase my skills, projects, and contact information in a professional and visually engaging way. The website is fully responsive and works on desktop, tablet, and mobile devices.

---

## ✨ Features

- 🎨 Colorful modern design with animated background
- 🌙 Dark / Light mode toggle (saved with localStorage)
- 📱 Fully responsive with mobile hamburger menu
- 🔢 Animated skill progress bars
- 💼 Dynamic project cards loaded from JavaScript data
- ✅ Contact form with real-time validation
- 🔗 Smooth scroll navigation

---

## 🛠️ Technologies Used

| Technology | Usage |
|------------|-------|
| HTML5 | Page structure and semantic elements |
| CSS3 | Styling, animations, Flexbox, Grid, Media Queries |
| JavaScript | Interactivity, DOM manipulation, localStorage |
| Google Fonts | Typography (Syne + DM Sans) |
| GitHub Pages | Deployment and hosting |
